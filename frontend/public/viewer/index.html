SMR TwinMaker Viewer – External Bundle Placeholder

This is a placeholder page to host a prebuilt TwinMaker scene viewer bundle. To enable the embedded viewer without installing heavy packages in this environment, build the viewer externally (or use CookieFactoryV3 sample) and drop the compiled assets into public/viewer/.

  1. Clone AWS sample: aws-iot-twinmaker-samples
  2. Build the CookieFactoryV3 dashboard or a minimal AppKit scene viewer on your machine.
  3. Copy the built index.html, JS and CSS bundles into this folder (public/viewer/).
  4. Reload the dashboard – the iframe will render your viewer here.

If you already placed files here and still see this message, ensure index.html exists and loads its JS bundles with relative paths.
